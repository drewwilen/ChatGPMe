# Minimal Colab Upload

This repo can generate a minimal package for Colab so you do not need to upload the full project.

The package contains only:

- `train_mvp.jsonl`
- `train_lora.py`
- `generate_with_adapter.py`
- `requirements.txt`
- `README.md`

After generating it, upload `colab_upload.zip` to Google Drive or directly into Colab and run from there.
